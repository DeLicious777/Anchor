# Agents

Reusable subagent definitions for this project's multidisciplinary team:

- [product-manager.md](product-manager.md) — vision, users, MVP scope, prioritization
- [ux-designer.md](ux-designer.md) — user flows, interaction design
- [technical-architect.md](technical-architect.md) — technology decisions, system structure, ADRs
- [senior-software-engineer.md](senior-software-engineer.md) — implementation feasibility, effort reality-checks
- [researcher.md](researcher.md) — market/technical research, evidence gathering
- [reviewer.md](reviewer.md) — challenges assumptions, runs `/grill-with-docs` before docs are finalized
- [documentation-steward.md](documentation-steward.md) — keeps docs synchronized and drift-free

Invoked via the Agent tool by name, or automatically when a task matches an agent's `description`.
