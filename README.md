# Anchor

An AI-first product workspace. This repository is in **discovery** — no finalized idea, architecture, tech stack, or roadmap yet. See the [Definition of Ready](docs/README.md) for what must exist before implementation begins.

## Structure

| Path | Purpose |
|---|---|
| `docs/` | Living documentation — vision, concept, product, architecture, ADRs, assumptions, risks, roadmap, glossary |
| `research/` | Market/technology research and investigation |
| `ideas/` | Raw, unfiltered ideas before triage |
| `prototypes/` | Throwaway experiments — never production code |
| `planning/` | Epics, milestones, issue conventions — populated once design/architecture are ready |
| `graphify-out/` | Committed knowledge graph output |
| `.claude/` | Reusable agents, workflows, commands, and prompts for AI-assisted development |

## Working style

No implementation until the [Definition of Ready](docs/README.md) is met. Decisions are recorded as ADRs in `docs/decisions/`, numbered sequentially. See [CLAUDE.md](CLAUDE.md) for project-specific AI collaboration rules.
