# Ideas

Inbox for raw, unfiltered ideas before triage. Nothing here is a commitment.

- One file per idea, no required structure — capture first, refine later.
- When an idea matures, promote it into `docs/product/features/` (as a feature doc) or `docs/decisions/` (as an ADR), and remove it from here.
- It's fine for ideas to die here. Not everything needs to graduate.
